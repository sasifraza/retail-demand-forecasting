# Retail Demand Forecasting

This project demonstrates how machine learning can be used to forecast retail product demand.

Key components
• Synthetic retail sales dataset
• Feature engineering using lag demand
• Random Forest regression model
• Demand visualization and forecasting

Tools
Python, Pandas, Scikit-Learn, Matplotlib

Goal
Help retailers anticipate demand and improve inventory planning.

## Example Output
The Random Forest model was trained on synthetic retail demand data
with lag features and calendar variables.

Evaluation metric:
MAE (Mean Absolute Error) reported in the notebook.

The model demonstrates how machine learning can be applied to
retail demand forecasting and inventory planning.
